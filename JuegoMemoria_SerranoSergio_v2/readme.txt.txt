El programa memorin es una página que pide al usuario un número de filas y de columnas, despues de insertarles dichas filas y columnas el programa creará una tabla con tantas filas y columnas hallamos pedido y llenará la tabla con parejas de números.

IMPORTANTE!!
no se creará la tabla en los siguientes casos:
- si el número de casillas es impar
- si solo hay dos casillas
- si el número de casillas es 0 o inferior a 0
- si se introducen valores que no sean números enteros (tales como decimales,letras,    carácteres especiales, etc).
